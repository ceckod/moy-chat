Статистика — YouTube Тракер (GitHub Actions)

Дневен, автоматичен tracker живее извън браузъра — GitHub Actions cron сам се събужда всеки ден, дърпа статистика за целия канал (абонати, views, likes, comments за всяко видео) и я записва в data/stats-history.json в repo-то. Таблото само чете този файл.

Еднократен setup в GitHub (виж README.md в repo-то за пълни стъпки)
  1. Добави YOUTUBE_API_KEY като GitHub Secret (Settings → Secrets → Actions).
  2. Сложи своя Channel ID в config.json в repo-то.
  3. Workflow-ът в .github/workflows/daily-stats.yml тръгва сам всеки ден.
